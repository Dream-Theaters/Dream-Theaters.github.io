一.对数据库的操作（增删改查）
–创建数据库：create database 数据库名；

–删除数据库 drop database 数据库名；

–修改数据库 alter database 数据库名 harset=gbk；

–查询所有数据库：show databases；

–查看建立数据库的语句：show create database 数据库名；
show databases like ‘%t%’;
–like命令，显示所有名字里带t的数据库名
–%匹配0个或多个任意字符，_ 匹配任意一个字符

–设置默认数据库 use 数据库名；

二.对数据表的操作
–创建数据表（增删改查）

–删除表
drop table 表名；

–修改表
alter table 表名 engine=innodb；–可以修改默认引擎
alter table 表名 charset=gbk；–可以修改默认字符集
rename table 表名1 to 表名2；–修改表名

–查看表
show tables；查看所有数据表
show create table 表名；查看建表语句
desc 表名；查看表结构

三.对字段的操作
创建字段在创建数据表时进行，在已经拥有了表之后，还可以对字段进行操作

–增加新字段

–删除字段

–修改字段类型和属性

–修改字段的名字和定义

四.对记录的操作
–新增记录：insert into 表名（字段列表）values （值的列表）；
–注意：值必须与字段列表一一对应
–值得个数与数据表中字段的个数完全一致时，可以省略字段列表

–记录查询
select [字段列表] [from子句] [inner join子句] [where 子句] [group by子句] [having子句] [order by子句] [limit 子句][字段列表]：字段1，字段2，字段3......
* :通配符，匹配所有字段
-- select username from student; –查看所有人的姓名
-- select username,age from student; –从学生表中查看所有人的姓名和年龄

–from子句：用于为select语句提供数据，确定数据源
1个表：将这个表设置为数据的来源表
多个表：联表查询
还可以为表格指定别名
–从学生表中查询出所有数据 select * from student；
–指定别名为s select *from student s；
–从学生表和班级表中获取数据 select * from student，classes；

–where子句：用于过滤数据，只取出满足条件的记录

–order by子句：用于对显示的数据进行排序
–select * from student where age>=20 order by age asc;
–select * from student where age>=20 order by age desc;

–limit子句：用于限制输出数据的条数
–查询出年龄大于20岁的同学，只要最小的两个。

limit n; –提取前n条数据
limit m，n；–从m条开始提取n条数据

–修改记录
update 表名 set 字段名1=新的值1，…字段名n=新的值n，where子句；
–将所有同学都归为ID是1的班级 update student set classid=1；
update student set age=22 where username=“张三”；

–删除记录
delete from 表名 where 条件子句；
delete from student where age=0；--删除年龄为0的所有同学
